Vamos Documentation
===================

* See http://hyperpolyglot.org/lisp for comparison of different Lisps.

Special Forms
-------------

(def x 4)

(fn (x y z) (+ x (+ y z)))

(quote x) --OR-- 'x

(if <BOOL> <THEN> <ELSE>)

Built-in Functions
------------------

Math: +, -
Logical: =, <, >

Boolean Values
--------------

False values: false (the symbol), which is also stored in false (the variable)
True values: true (the symbol), also stored in true (the variable)
