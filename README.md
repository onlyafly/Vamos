## What is Vamos?

Vamos is a Lisp written in Go.

For interesting reading on Lisps, see https://github.com/onlyafly/lisp-bibliography

## Usage

### Running Test Suite

    $ make test

### Building From Source

    $ make

### Start REPL

    $ vamos

### Load forms from file then start REPL

    $ vamos -l foo.v
